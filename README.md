Репозиторий содержит код визуализации, использованный при подготовке выпускной квалификационной работы на тему «Влияние показателей нефинансовой отчётности на стоимость российских компаний добывающего сектора».

## Содержание репозитория

- `graphs.py` — Python-скрипт для построения графических материалов исследования.
- `requirements.txt` — список библиотек Python, необходимых для запуска кода.

## Назначение кода

Код используется для построения графических материалов эмпирической части исследования, включая:

- карту пропущенных значений;
- корреляционную матрицу;
- графики распределений;
- графики коэффициентов и доверительных интервалов;
- дополнительные графики для проверки устойчивости результатов.

## Входные данные

Для запуска скрипта требуются входные файлы в формате CSV:

- `data.csv`;
- `model_coefficients.csv`.

Исходный авторский датасет не размещается в открытом доступе. Он был сформирован вручную на основе финансовой и нефинансовой отчётности компаний, данных Московской биржи, СПАРК-Интерфакс и открытых статистических источников. Первичные источники данных и порядок формирования переменных описаны в тексте ВКР.

## Используемые инструменты

Код написан на Python 3.12 с использованием библиотек:

- pandas;
- numpy;
- matplotlib.
